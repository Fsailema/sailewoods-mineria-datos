# 📊 Minería de Datos - SaileWood's

## Descripción
Este proyecto analiza los datos de ventas de productos de madera para identificar patrones y mejorar la toma de decisiones.

## Objetivo
Aplicar técnicas de minería de datos para segmentar productos según su comportamiento de ventas.

## Dataset
Archivo Excel con información de productos, precios y cantidades vendidas.

## Tecnologías
- Python
- Pandas
- Matplotlib
- Scikit-learn

## Estructura del proyecto
- data: dataset
- notebooks: análisis exploratorio
- src: modelo
- resultados: gráficos y resultados

## Resultados
Se identificaron diferentes grupos de productos según su precio y nivel de ventas.
